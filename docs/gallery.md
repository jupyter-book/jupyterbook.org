# Gallery of Jupyter Books
Jupyter Book technology has been applied across a wide range of use cases, including curricula vitae, official educational textbooks, student portfolios, lab manuals, and technical documentation.
This is a gallery of Jupyter Books built from across the community.
For interactive geoscience cookbooks, see also the [Project Pythia Cookbook Gallery](https://cookbooks.projectpythia.org).
To add your book to this list, [add an entry to this gallery.yml file](https://github.com/jupyter-book/jupyterbook.org/edit/main/docs/src/gallery.yml) and open a Pull Request.
